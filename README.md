# Online Learning Platform

A Spring Boot based Online Learning Platform web application.

# Technologies Used
- Java Spring Boot
- Maven
- HTML, CSS, JavaScript

# How to Run
1. Clone the repository
2. Run: `./mvnw spring-boot:run`
3. Open browser: `http://localhost:8080`

# Project Structure
- `HomeController.java` - Handles routes
- `OnlineLearningPlatformApplication.java` - Main entry point
- `templates/` - HTML pages
- `static/` - CSS, JS, Images
